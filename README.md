# CircularProgressTimer

A circular progress countdown view.

Usage:

CPTCountdownView(min: 0, max: 30)
      .clockSize(CGSizeMake(300, 300))
      .borderWidth(10)
      .fontColor(.red)
      .trackColor(.blue)
      .barColor(.pink)
      .completionColor(.purple)
      .clockFont(.system(size: 70))
