# CircularProgressTimer

A circular progress countdown view.

Usage:

```Swift
CPTCountdownView(min: 0, max: 80)
      .clockSize(CGSizeMake(300, 300))
      .borderWidth(18)
      .fontColor(.black)
      .trackColor(.gray.opacity(0.2))
      .barColor(.orange)
      .completionColor(.green)
      .clockFont(.system(size: 70))
```
